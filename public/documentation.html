MyFlix API

Version 1.0.0


Users are able to review movie details with this web application. This web application allows users to access genres, directors, release dates, and concise descriptions for a seamless cinematic experience. In addition, users are able to create and update their account with a list of their favorite movies!

Return genre data by name /movies/genre/:genreName GET/READ None A JSON object holding data about a single genre, containing genre name, description.
