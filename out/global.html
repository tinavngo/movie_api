Movie schema Defines the structure of a movie document, including details such as title, description, MPAARating, etc.
Properties:
Name Type Description
Title string The title of the movie.
Description string The description or summary of the movie.
MPAARating string The MPAA rating of the movie (e.g., PG-13, R).
ReleaseYear string The year the movie was released.
Genre Object The genre of the movie, including name and description.
Director Object The director of the movie, including name and bio.
Actors Array.<string> An array of actors featured in the movie.
ImagePath string URL or path to the movie's image.
Featured boolean Boolean indicating if the movie is featured.
